LDE Open WebUI
Add Open WebUI service to the LDE deployment for testing and interacting with the Hugr mcp service.
Open WebUI repository: https://github.com/open-webui/open-webui
Tasks:
1. Add Open WebUI service to the docker-compose.yml with port mapping to usage range (in the same range as other services 19000-19020)
2. Add environment variables to the .env.example
3. Add information to the lde readme and in the main readme about the Open WebUI service in the LDE
4. Add information to the help scripts to setup and start the Open WebUI service
